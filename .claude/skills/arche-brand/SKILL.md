# Skill — archè brand discipline

## Identità in una frase
"Dal mito alla mente": ogni crimine è la replica di una tragedia
già scritta. archè è il brand-media che mostra l'origine; Gabriella
Marano è la voce. Estetica: museo archeologico, non true-crime.

## Lessico ammesso
mito, archetipo, origine, tragedia, voce, lente, traccia, stratigrafia,
matrice, episodio, gesto, figura, scena, atto, soglia, restituzione,
custode, archivio.

## Lessico vietato
caso della settimana, crimine scioccante, breaking news, cronaca nera,
dietro le quinte del male, scoop, segreti, esclusivo, true crime,
killer, mostro, raccapricciante, agghiacciante, esclusiva.

## Copy — checklist
Prima di consegnare qualunque testo pubblico (titoli, sottotitoli,
meta description, CTA, email, copy social), verifica:

- [ ] Nessun nome di vittima reale di omicidio recente
- [ ] Nessun nome di indagato in indagine aperta
- [ ] Nessun riferimento a casi di cronaca dell'ultimo anno
- [ ] Nessuna diagnosi su persona reale
- [ ] Nessun aggettivo emotivo ("agghiacciante", "scioccante")
- [ ] Nessun cliché true-crime ("dietro le quinte", "verità nascosta")
- [ ] Tono asciutto, autorevole, non urgente
- [ ] Se cita un caso reale, lo fa solo come contesto di un'apparizione
      editoriale (es. "Intervenuta a Quarto Grado sul tema X"), mai
      come materia del prodotto

Se anche una voce fallisce: riscrivi, non consegnare.

## Visivo — checklist
- [ ] Mai immagini di scene del crimine reali
- [ ] Mai immagini di vittime reali identificabili
- [ ] Mai rosso-sangue (oxblood #2A1412 è il massimo, raro)
- [ ] Mai gradienti pacchiani, glow neon, lens flare
- [ ] Consentiti: arte greca antica, statuaria classica, teatro greco,
      manoscritti, reperti, affreschi, ceramiche figurate, marmo
- [ ] Vietati: cinema true-crime, polizieschi TV, collage di volti,
      ritratti segnaletici, scene di tribunale

## Tipografia — regole inderogabili
- Display: Bodoni Moda (titoli, hero, h1-h2)
- Body: Spectral (paragrafi, lead, testo)
- Secret (Cormorant Garamond) SOLO in: numeri di sezione,
  titoli di voci, maiuscolette museali
- Sans-serif: solo eyebrow (.eyebrow) e UI tecnico minimo
- Mai Inter, Roboto, Comic Sans come display

## Palette — applicazione
- Sfondi: ink/ink-2/ink-3 — sempre scuri
- Testo: bone/bone-dim/bone-faint
- Accenti: gold, mai più del 12% della superficie visiva
- Linee: rgba oro o osso, mai pure
- Errori/warning: oxblood o gold, mai rosso vivo

## Naming codice
Variabili e funzioni in inglese. Entità di dominio mantengono
parole italiane intraducibili nelle route: /voce, /matrice.

## Quando applicare
- PRIMA di scrivere copy pubblico
- PRIMA di scegliere immagini o generare illustrazioni
- PRIMA di nominare nuove route o sezioni
- PRIMA di scrivere meta description, alt text, social copy
- DOPO ogni copy generato: rilettura con checklist
